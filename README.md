# Taller 01 - Procrastinación (ICCI)

Este programa gestiona los tiempos de ocio y actividades de un grupo de usuarios, leyendo desde archivos de texto.

### Integrantes
* Vicente Garriga - 22.380.392-k - [vicentegarriga-stack]
* Martin Alvarado - 22.330.833-3 - [Martinooski]

### Estructura
Lógica centralizada en clase Main.java utilizando arreglos paralelos para el almacenamiento en memoria.

### Instrucciones
1. Ejecutar Main dentro de un package llamado "taller01".
2. IMPORTANTE: El programa requiere que existan los archivos Usuarios.txt y Registros.txt en la misma carpeta raíz del proyecto para poder cargar la información.
